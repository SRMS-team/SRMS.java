# Student Recored  Managment 
##1. How to Compile
Open the command line and navigate to the project directory:
```bash
javac *.java
```
##2. How to Run
```bash
java Main
```
##3. How to Test

1. Compile and run the project.
2. When the program asks for input, enter test values using the keyboard.
3. Observe the output displayed in the command line.
4. Compare the output with the expected results.
5. Test the program with multiple cases to verify accuracy.
